# SeT-LLM 2026 — Secure and Trustworthy Large Language Models | KDD Workshop

This repository hosts the website for **SeT-LLM 2026**, a workshop on secure and trustworthy large language models at **KDD 2026** in Jeju, Republic of Korea.

## About

SeT-LLM 2026 addresses the security, trustworthiness, and safety challenges of LLM-powered systems. The workshop covers topics such as reliability assurance and evaluation, privacy leakage and copyright protection, interpretability and plagiarism detection, secure deployment, adversarial attacks and defenses, toxic content mitigation, and new learning and interaction paradigms for LLM systems.

The workshop is scheduled for **9–13 August 2026** on **Jeju Island, Republic of Korea**.

## Website Structure

- `index.html` — main landing page with sections for Home, About, Call for Papers, Schedule, Speakers, and Organizers
- `assets/css/style.css` — main stylesheet and section styling
- `assets/js/main.js` — template JavaScript for navigation and scrolling behavior
- `assets/vendor/` — third-party libraries used by the site (Bootstrap, AOS, GLightbox, Swiper, etc.)
- `assets/img/` — site images, favicons, and organizer photos
- `accepted_papers.csv` — accepted paper data used by the site when the accepted papers section is enabled

## Notes

The current site is built from a Bootstrap template and styled for the SeT-LLM 2026 workshop. The call for papers section includes important dates, submission guidelines, and workshop topics, while the schedule section provides a provisional agenda.
